# gl-inet-mt3600be-passwall-switch

## 🤔 这是什么？

这个项目可以快速设置 MT3600BE 路由器的快捷开关（Toggle Button）来控制启停 passwall.
请确保已经安装 passwall!

## 🚀 快速上手

### 1. SSH连接到路由器,或者在路由器终端执行如下命令

```bash
wget -O gl-inet-mt3600be-passwall-switch-install.sh https://github.com/jinzhijiang/gl-inet-mt3600be-passwall-switch/releases/download/1.0/gl-inet-mt3600be-passwall-switch-install.sh && chmod +x gl-inet-mt3600be-passwall-switch-install.sh && ./gl-inet-mt3600be-passwall-switch-install.sh
```

### 下次如何调用,在当前目录下执行
```bash
sh gl-inet-mt3600be-passwall-switch-install.sh
```
